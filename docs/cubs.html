Web app by Tim Broderick tbroderick@dailyherald.com.
I'm following the baseball season by developing this app, written in python/flask and utilizing the pybaseball package by James LeDoux. The goal is to learn more python and baseball statistics as I gradually add features and content. All my code so far can be found here. At some point, I'll set this up to update automatically each day.
